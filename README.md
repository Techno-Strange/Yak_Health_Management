# 🐂 Yak Health Management System

> A modern web-based platform for managing Yak Vaccination, Breeding, and Health Records, developed to support herders, veterinarians, researchers, and farm managers in high-altitude yak-rearing regions.

---

## 📖 Overview

The Yak Health Management System is a full-stack web application designed to digitize and streamline yak herd management. Traditional record-keeping methods often rely on paper registers and spreadsheets, making it difficult to track vaccination schedules, breeding history, and health records efficiently.

This system provides a centralized platform where users can securely store, retrieve, and manage yak-related data through an intuitive web interface.

---

## 🚀 Key Features

### 🐂 Yak Profile Management

* Create and manage individual yak records
* Store Yak Animal Number, age, sex, photographs, and remarks
* Quick search using Yak Animal Number

### 💉 Vaccination Record Management

* Track vaccination history
* Store vaccine names, dosage, and administration dates
* View complete vaccination records for each yak

### 🧬 Breeding Record Management

* Maintain breeding history
* Track service records and reproductive information
* Support breeding analysis and decision-making

### 🔍 Advanced Search & Filtering

* Search yaks using Animal Number
* Filter records efficiently
* Retrieve detailed health and breeding information instantly

### 📄 PDF Report Generation

* Generate printable yak reports
* Export detailed animal information

### 🔐 Secure Authentication

* User login and access control
* Role-based access for administrators, researchers, and veterinarians

### 📊 Dashboard & Analytics

* Visual overview of herd information
* Recent activity monitoring
* Improved decision-making through organized data

## 🎯 Problem Statement

Yak herders often rely on manual record-keeping systems that are difficult to maintain, prone to errors, and challenging to access remotely.

The Yak Health Management System solves this problem by providing:

* Digital record management
* Improved data accuracy
* Easy access to historical records
* Better collaboration among herders, veterinarians, and researchers
* Scalable cloud-based storage
